Html stands for Hyper Text Markup Language and is mainly used to create documents on the web. A physicist in 1980 came up with an idea of a language that could be used to create web pages so he followed through and created it and called it html.It can be writen in many applications and progams found on the internet. It is written with a series of "tags" and commands.There is another coding language called css or Cascading style sheet.This is more of a decorating language.It is used in addition with HTML. It was created when a man wanted to add more details to his html webpage.It is written with a series of "spans" that decribe and give attributes to the HTML.A tag is a command that needs a start, ending, and info in the middle.You would need to start with the greater than symbol, then your command, and then a less than symbol. You would end it with the same thing but instead you add a backslash before the command. Attributes are used to change the looks of your webpage.Some examples are if you would want to change the font, font size, colors, borders,and designs. Some examples of different tags would be if you wanted to write text you would need a "p" tag. Also an "ol" tag is used to make an ordered list, a "em" tag is used to put emphasis on a word or phrase, and a "img" tag is used to insert an image.
